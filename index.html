Terrarium pour plantes carnivores

Plantes :

  • Heliamphora : heterodoxa x minor, ciliata, nutans, ionasii
  • Cephalotus : follicularis

Conditions optimales pour Heliamphora :

  • Substrat très humide
  • Écart de température jour/nuit de 10°C
  • Fort éclairage
  • Humidité de l'air de 80 %

Éclairage :

  1. Deux modules LED blanc : contrôle via une prise domotisée ZigBee
  2. Panneau LED bleu/rouge : contrôle via ESP32 (led infrarouge connecté sur GPIO simulant une télécommande)
Plantes carnivores, Terrarium, Domotique
Terrarium pour escargots

Archachatina marginata ovum : espèce d'escargot la plus grosse du monde.

  • Terrarium fabriqué : vitres de recyclage, liège isolant, contreplaqué, charnières imprimées en 3D
  • Régulation : Thermostat DIY avec plaques chauffantes + système de ventilation (gestion de température / humidité)
  • Lumière : L'escargot ne nécessite pas de lumière. Les LEDs sont purement esthétiques et sont contrôlées via une prise domotisée ZigBee.
